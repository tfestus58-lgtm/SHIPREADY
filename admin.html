Flutterwave Payout Minimums

Minimum bank withdrawal per currency, in that currency's own units — not USD. Every payout in these currencies is a real NGN→X conversion from Flutterwave's side (even a "same-currency-looking" GHS→GHS payout), so it always carries a live Flutterwave fee. Set each minimum high enough that the fee never eats an unreasonable share of a withdrawal. NGN is excluded — it's Flutterwave's settlement currency, so NGN payouts never incur this fee.

Based on Flutterwave's published fees as of implementation — not a permanent guarantee. Flutterwave's fees can drift over time, so revisit these periodically.
